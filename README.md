
# My first django project (blog)

---

### How to install it.

* Create virtualenv.
* Clone the project
* Install requirements.
* Run the project.

## Commands to run the project.

On Windows.

```
cd Downloads
mkdir django_site
python -m venv venv
cd venv/Scripts
activate 
../../
git clone https://github.com/tiberius-kirk/My_django_blog.git
pip install -r requirements.txt
```
